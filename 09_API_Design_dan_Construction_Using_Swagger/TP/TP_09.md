## Deskripsi Program
Program ini adalah API sederhana yang dibuat menggunakan Node.js dan Express. Fungsinya nggak ribet, yaitu menyediakan endpoint GET /menu untuk menampilkan daftar kategori menu yang tersedia.

Saat endpoint localhost:3000/menu diakses, server akan mengirimkan data dalam format JSON. Data yang dikirim berisi daftar kategori menu yang sudah disimpan di dalam program, yaitu "bakmi" dan "rames".

Jadi, siapa pun yang membuka endpoint tersebut lewat browser atau mengirim request menggunakan Postman akan langsung mendapatkan daftar kategori menu tersebut tanpa perlu melakukan proses tambahan apa pun.
